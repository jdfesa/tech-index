# 🌟 Tech Index & Resources

A curated repository of links, tools, and resources with a premium glassmorphic palette, inspired by modern "Awesome" lists.

Now supports **Dark Mode** and **Soft Paper Light Mode**!

---

## 🚀 Quick Start

### 1. Clone & Install Dependencies
```bash
git clone https://github.com/jdfesa/tech-index.git
cd tech-index
npm install
```

### 2. Run Local Development Server
```bash
npm run dev
```
Open `http://localhost:5173` to preview the interface in your browser in real-time.

---

## 🛠️ Built With

- **React:** For dynamic component rendering and state management.
- **Vite:** Instant Hot Module Replacement (HMR) for ultra-fast development.
- **Vanilla CSS:** Pure modular styles without heavy frameworks. Optimized using CSS Variables (`:root`), glassmorphism effects (`backdrop-filter`), and animated border gradients.
- **Lucide React:** A beautiful collection of clean, vector iconographies.

## 🤝 Community & Contributing

Contributions are heavily welcomed to make this index grow! 

If you find a tool, guide, or workflow useful, you can add it **without writing a single line of React code**:
1. Simply open and edit `src/data/resources.json`.
2. Append your resource entry to the `"resources"` list following the template.

👉 For detailed formatting rules on adding items, please check the **[Contributing Guidelines](./CONTRIBUTING.md)** file! No Pull Request is too small. Let's build a rich knowledge index together. 💛
